# Favorite Problems

This folder contains problems that I personally find interesting, elegant, memorable, or meaningful.

A problem does not have to be hard to be here.  
It only needs to teach me something useful or feel satisfying to solve.

## Why this folder exists

Some problems are worth remembering because they show a clean idea, a nice trick, or a pattern that appears again and again.

This folder is for those problems.

## Possible Reasons

A problem may be added here if:

- the idea is elegant
- the implementation is clean
- I made an interesting mistake
- it reminds me of competitive programming
- it shows a useful Python trick
- I simply like the problem

## Examples

- Container With Most Water
- Longest Substring Without Repeating Characters
- Palindromic Substrings
- Word Break
- Trapping Rain Water
- Minimum Window Substring
