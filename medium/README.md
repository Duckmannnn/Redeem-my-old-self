# Medium Problems

This is the main folder of the repository.

Medium problems are where I can actually rebuild my problem-solving rhythm. These problems are usually not too hard, but they require recognizing the right pattern and implementing it cleanly.

## Purpose

- Practice common LeetCode patterns
- Convert CP-style thinking into Python
- Improve implementation speed
- Rebuild confidence with algorithmic problems

## Common Topics

- Two pointers
- Sliding window
- Hash map
- Prefix sum
- Intervals
- Backtracking
- Dynamic programming basics
- Graph traversal

## Examples

- Longest Substring Without Repeating Characters
- Longest Repeating Character Replacement
- Container With Most Water
- Merge Intervals
- Insert Interval
- Non-overlapping Intervals
- Permutations
- Palindromic Substrings
- Word Break

## Note

For normal medium problems, I only save the solution.

For problems that teach me something useful, I may add a short comment at the top of the file.
