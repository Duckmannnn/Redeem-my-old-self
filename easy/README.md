# Easy Problems

This folder contains easy problems that help me warm up with Python.

Most of these problems are not algorithmically difficult for me, but they are useful for getting used to Python syntax, built-in data structures, and cleaner implementation.

## Purpose

- Practice Python basics
- Rebuild consistency
- Get comfortable with LeetCode format
- Avoid overthinking simple problems

## Examples

- Two Sum
- Valid Parentheses
- Valid Anagram
- Contains Duplicate
- Best Time to Buy and Sell Stock
- Binary Search
- Roman to Integer

## Note

I usually do not write detailed notes for easy problems unless there is a small Python trick or implementation detail worth remembering.
