# Hard Problems

This folder is for difficult problems that require stronger observation, cleaner implementation, or deeper algorithmic thinking.

I do not expect this folder to grow quickly.  
Hard problems are not the main focus at the beginning, but they are useful milestones.

## Purpose

- Challenge myself with harder problems
- Rebuild stronger CP-style thinking
- Practice patience with complex implementation
- Save solutions that mark real progress

## Example Future Problems

- Trapping Rain Water
- Minimum Window Substring
- Sliding Window Maximum
- Binary Tree Maximum Path Sum
- Median of Two Sorted Arrays

## Note

For hard problems, I may add short notes explaining the main idea, especially if the solution feels elegant or difficult to rediscover.
